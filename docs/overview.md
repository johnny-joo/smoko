# 담배 피러가기 🚬 — 프로젝트 개요

> 10분간의 달콤한 현실 도피. 집중보다 **쉬는 시간**에 초점을 맞춘 뽀모도로 스타일 휴식 시뮬레이션.

---

## 컨셉

직장인이 담배를 피러 나가는 루틴을 그대로 시뮬레이션한다.
카페 뽀모도로 ([cafe.pauwee.com](https://cafe.pauwee.com/)) 처럼 심플한 UI 속에서 몰입감 있는 경험을 제공하는 것이 목표.

```
편의점 담배 선택 → 엘리베이터 탑승 → 흡연 정원 도착 → 담배 불 붙이기 → 연기 뱉으며 10분 휴식
```

---

## 파일 구조

```
smoke-break-pomodoro/
├── index.html        # 전체 씬 마크업 (5개 씬 + 인트로/엔딩)
├── style.css         # 모든 씬 스타일, 애니메이션, 반응형
├── app.js            # 씬 전환 로직, 타이머, 파티클 시스템, 상태 관리
├── assets/           # 이미지·영상 에셋 (현재 비어있음 — AI 생성 예정)
└── docs/
    ├── overview.md         ← 이 파일
    ├── scenes.md           # 각 씬 상세 구현 내용
    ├── todo-detail.md      # 추가 구현이 필요한 항목 정리
    └── ai-assets-guide.md  # AI 이미지·영상 생성 가이드
```

## 기술 스택

| 항목 | 내용 |
|------|------|
| 언어 | HTML5 · CSS3 · Vanilla JS (ES2020) |
| 폰트 | Google Fonts — Noto Sans KR, Black Han Sans |
| 그래픽 | CSS 애니메이션 + Canvas 2D (연기 파티클) |
| 외부 의존성 | 없음 (CDN 폰트 제외) |
| 빌드 도구 | 없음 — 파일 직접 실행 가능 |

---

## 실행 방법

1. `smoke-break-pomodoro/` 폴더를 VS Code Live Server 또는 아무 HTTP 서버로 열기
2. `index.html`을 브라우저에서 직접 열어도 동작함 (파일 프로토콜 허용)
3. 모바일에서도 접근 가능 (반응형 대응)

## 씬 흐름

```
[인트로] ──▶ [편의점] ──▶ [엘리베이터] ──▶ [흡연 정원] ──▶ [흡연 중 타이머] ──▶ [엔딩]
                                                                        ▲
                                                              스페이스바 / 클릭으로
                                                                연기 내뱉기
```
