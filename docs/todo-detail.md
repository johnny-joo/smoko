# 추가 구현 필요 항목

현재 구현은 기능적으로 완성된 MVP입니다.  
아래는 더 디테일하고 완성도 높은 서비스를 위해 추가할 수 있는 항목들입니다.

---

## 🔴 높은 우선순위 (경험 품질에 직결)

### 1. 배경음악 & 효과음
가장 몰입감에 영향이 큰 요소. cafe.pauwee.com도 오디오가 핵심.

| 구간 | 권장 사운드 |
|------|------------|
| 편의점 씬 | 편의점 BGM (잔잔한 J-Pop 계열) |
| 엘리베이터 씬 | 엘리베이터 작동음, 승강기 BGM |
| 흡연 정원 | 야외 자연음 (새소리, 바람) |
| 흡연 중 타이머 | Lofi 힙합 or 빗소리 앰비언트 |
| 라이터 클릭 | 딸깍 + 불꽃 효과음 |
| 연기 뱉기 | 후~ 숨 내뱉는 효과음 |

**구현 방향**
```js
// Web Audio API 또는 Howler.js 라이브러리 사용
// assets/audio/ 폴더에 .mp3 / .ogg 파일 배치
// 사운드 온/오프 토글 버튼 UI 상단에 추가
```

---

### 2. AI 생성 이미지 교체 (assets/ 폴더)
현재는 CSS/이모지로 표현된 배경을 실제 이미지로 교체하면 완성도 대폭 상승.
→ **별도 가이드 문서**: `docs/ai-assets-guide.md` 참고

---

### 3. 엘리베이터 씬 배경 영상
현재는 CSS만으로 구현. 실제 엘리베이터 내부를 내려다보는 영상(AI 생성 or 스톡)을 배경에 깔면 극적으로 몰입감 향상.

```html
<!-- index.html scene-elevator 내부에 추가 -->
<video autoplay muted loop playsinline id="elev-video" class="elev-bg-video">
  <source src="assets/elevator.mp4" type="video/mp4" />
</video>
```

## 🟡 중간 우선순위 (디테일 & 재미 요소)

### 4. 담배 종류 추가 / 커스텀 담배
- 현재 6종 → 10종 이상으로 확장
- "나만의 담배" 커스텀 기능: 이름, 컬러 직접 입력
- 담배별로 타이머 길이를 다르게 (순한 담배 = 짧게, 독한 담배 = 길게)

### 5. 날씨 연동 흡연 정원
- Open-Meteo API (무료)로 현재 날씨 가져오기
- 맑으면 파란 하늘, 비오면 빗소리 + 비 파티클, 흐리면 구름 많이
```js
// 예시
const weather = await fetch('https://api.open-meteo.com/v1/forecast?latitude=37.5&longitude=127&current_weather=true');
```

### 6. 연기 파티클 개선
- 현재: 단순 원형 그라디언트
- 개선: 실제 담배 연기처럼 회오리 모양, 더 복잡한 turbulence 효과
- WebGL (Three.js / pixi.js) 으로 업그레이드 시 훨씬 사실적 표현 가능

### 7. 담배 재 떨기 인터랙션
- 긴 재가 쌓이면 "탁탁" 클릭해서 재떨이에 터는 인터랙션 추가
- 재 파티클 낙하 애니메이션

### 8. 엔딩 공유 기능
- "오늘의 휴식 기록" 이미지 생성 → `canvas.toDataURL()`로 PNG 다운로드
- 카카오톡 공유 / 트위터 공유 버튼

## 🟢 낮은 우선순위 (완성도 polish)

### 9. 로컬스토리지 히스토리
- 휴식 기록 저장 (날짜, 담배, 시간, 내뱉기 횟수)
- "이번 주 휴식 현황" 미니 대시보드

### 10. 타이머 길이 커스텀
- 기본 10분 외에 5분 / 15분 선택 가능
- 인트로 또는 편의점 씬에서 선택 옵션 제공

### 11. 가이드 애니메이션 (온보딩)
- 첫 방문 시 각 씬에서 무엇을 해야 하는지 손가락 포인터 애니메이션으로 안내
- `localStorage`로 첫 방문 여부 판별

### 12. PWA 지원
- `manifest.json` + 서비스 워커 추가
- 홈 화면에 추가 가능 (모바일 앱처럼 실행)
- 오프라인에서도 동작

### 13. 다크모드 외 테마
- "밤 흡연" 테마: 어두운 정원, 반딧불 파티클
- "옥상" 테마: 도심 야경 배경

### 14. 접근성 개선
- 모든 인터랙티브 요소에 `aria-label` 보강
- 키보드 전체 네비게이션 (Tab 키로 씬 진행 가능하도록)
- `prefers-reduced-motion` 미디어 쿼리로 애니메이션 축소 옵션

## 🐛 알려진 버그 / 개선 필요

| 항목 | 내용 | 해결 방향 |
|------|------|----------|
| 엘리베이터 재진입 | 이미 initElevator()가 실행된 상태에서 다시 진입 시 타이머 중첩 가능성 | setInterval 클린업 로직 추가 |
| 모바일 Canvas 해상도 | 고밀도 디스플레이(Retina)에서 연기가 흐릿할 수 있음 | `devicePixelRatio` 처리 추가 |
| 씬 오버플로우 | 일부 소형 기기(320px 이하)에서 편의점 씬 스크롤 필요 | 미디어 쿼리 추가 대응 |
| 브라우저 탭 숨김 | 탭이 백그라운드로 가면 `setInterval` 불정확 | `Page Visibility API` + `Date.now()` 기반 보정 |
