## Scene 0 — 인트로

**목적:** 사이트 진입점. 분위기 세팅.

**구현 내용**
- 떠다니는 🚬 아이콘 (CSS `@keyframes float`)
- 배경에 반투명 대형 텍스트 "SMOKE BREAK" 로 무드 연출
- "출발하기 →" 버튼 클릭 시 씬 1로 전환

## Scene 1 — 편의점 담배 선택

**목적:** 사용자가 오늘 피울 담배를 고른다. 소소한 선택권으로 개인화 경험 제공.

**구현 내용**
- 6종 담배 카드 그리드 (에쎄 체인지 / 말보로 레드 / 레종 블루 / 토브플러스 아이스 / L&M 실버 / 오늘의 특선)
- 카드에 브랜드별 컬러 그라디언트로 담배갑 디자인 표현
- 선택 시 `selected` 클래스로 테두리 하이라이트, 카운터에 이름·설명 표시
- 모바일 진동 피드백 (`navigator.vibrate`)
- 담배 미선택 시 "계산하기" 버튼 비활성화

**데이터 구조 (app.js CIG_DATA)**
```js
{
  esse:     { name, desc, color },
  marlboro: { name, desc, color },
  raison:   { name, desc, color },
  tobplus:  { name, desc, color },
  lm:       { name, desc, color },
  special:  { name, desc, color },
}
```

## Scene 2 — 엘리베이터

**목적:** 이동감을 주는 전환 씬. 숫자가 내려가는 연출로 몰입감 유지.

**구현 내용**
- 엘리베이터 내부 UI: 층수 디스플레이, 버튼 패널, 엘리베이터 문(좌/우)
- 층수 카운트다운: `11 → 9 → 7 → 5 → 3 → 1 → B1` (0.8초 간격)
- B1 도달 시 문이 양쪽으로 열리는 CSS `scaleX(0)` 트랜지션
- 선택한 담배 미니 프리뷰를 손에 들고 있는 것처럼 표현
- 문이 열리고 1.8초 후 자동으로 Scene 3 전환

## Scene 3 — 흡연 정원 & 불 붙이기

**목적:** 도착 후 라이터로 담배에 불을 붙이는 인터랙션. 클라이맥스 전 준비 씬.

**구현 내용**
- 배경: CSS로 하늘(파랑) + 잔디(초록) 그라디언트, 구름 드리프트 애니메이션, 식물 흔들림
- 담배 스틱: 필터(주황) + 몸통(흰색) + 끝부분(팁) CSS 조합으로 표현
- 라이터: 클릭 시 `.burning` 클래스로 불꽃 애니메이션 재생
- 불꽃이 붙으면 팁이 주황빛으로 변하고 `glow` 효과
- 진동 피드백 `[50, 30, 80]` 패턴
- 2.8초 후 자동으로 Scene 4 전환

## Scene 4 — 흡연 중 & 10분 타이머

**목적:** 핵심 씬. 실제 10분 휴식 타이머와 인터랙션.

**구현 내용**

### 타이머
- SVG 원형 링 타이머 (stroke-dashoffset 방식)
- `10:00` → `00:00` 카운트다운
- 남은 시간에 따라 링 색상 변화: 주황 → 노랑 → 빨강
- 타이머 만료 시 자동으로 Scene 5 전환

### 담배 시각화
- 실시간으로 담배 몸통이 줄어들고 재가 늘어나는 CSS width 트랜지션
- 끝부분 불씨 `emberPulse` 애니메이션

### 연기 파티클 (Canvas 2D)
- 매 프레임 자동으로 연기 입자 방출 (`requestAnimationFrame`)
- "후~ 내뱉기" 버튼 클릭 시 18개 burst 파티클 방출
- 파티클: 방사형 그라디언트, 위로 상승, 좌우 흔들림, 서서히 투명해짐

### 생각 말풍선
- 15개 직장인 생각 중 랜덤으로 8~20초마다 표시
- "연기 내뱉기" 클릭 시에도 즉시 새 생각 표시
- 중복 방지 로직 (`usedThoughts` 배열)

### 단축키
- `Space` 키로 연기 내뱉기 트리거

## Scene 5 — 엔딩

**목적:** 휴식 결과 요약. 따뜻한 마무리.

**구현 내용**
- 통계 카드: 내뱉기 횟수 / 실제 휴식 시간 / 선택한 담배
- 내뱉기 횟수에 따른 3단계 메시지 분기
  - 15회 이상: "연기를 엄청나게 내뱉었네요"
  - 8회 이상: "적당히 즐기셨군요"
  - 3회 이상: "잠깐이지만 충분한 휴식"
  - 3회 미만: "조용히 홀로 피우셨군요"
- "다시 피러가기" → 인트로로 리셋
- "업무 복귀" → 페이지 리로드
